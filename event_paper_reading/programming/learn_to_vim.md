## Skills

1. write a motion

```bash
# onoremap: o no recursive map
# Is: motion name
# : command mode
# <c-u> 消除符号
# normal: normal mode
# F]: 反向查找到]
# vi]: 选择[]中内容并进入visual mode
onoremap Is :<c-u>normal! F]vi]<cr>
```
